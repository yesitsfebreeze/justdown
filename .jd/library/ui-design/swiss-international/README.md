# Swiss / International Typographic Style

## Origin/Era

Swiss / International Typographic Style emerged in Switzerland in the 1940s–1960s, formalized by designers such as Josef Müller-Brockmann, Armin Hofmann, and Emil Ruder. It is built on mathematical grid systems and an ethic of objectivity and clarity — design as neutral, legible communication rather than personal expression. Its signature typefaces are the neutral grotesque sans serifs Akzidenz-Grotesk and, later, Helvetica.

## Defining traits

- Rigorous mathematical grid as the underlying structure for everything.
- Flush-left, ragged-right text (asymmetric typography).
- Neutral grotesque sans serif (Helvetica / Akzidenz-Grotesk).
- Strong typographic hierarchy via size and weight, not decoration.
- Generous whitespace as an active design element.
- Limited palette, often black + one accent (frequently red).
- Photography preferred over illustration.
- Objective, legible, free of ornament.

## When to use

- Information-dense interfaces where clarity and scanability matter (dashboards, docs, data tables).
- Editorial and publishing layouts that need a calm, authoritative voice.
- Corporate and institutional branding seeking neutrality and timelessness.
- Wayfinding and signage systems.
- Any product where the content, not the chrome, should be the focus.

## Pitfalls

- Can read as cold, corporate, or impersonal if applied without warmth.
- Rigid grids become sterile or monotonous without rhythm and contrast.
- Over-reliance on a single typeface demands real hierarchy discipline, or everything flattens.
- Generous whitespace can feel empty on small screens if not adapted.
- Easy to mistake "minimal" for "unfinished" — restraint still requires precision.
- Misuse of the red accent (too much, too saturated) destroys the calm balance.

## Token cheat-sheet

- Palette: black `#111111`, white `#FFFFFF`, swiss red `#D6001C`, neutral grey `#9B9B9B`.
- Border-radius: `0` everywhere — hard corners only.
- Shadow: none — flat surfaces; structure comes from grid + whitespace, not elevation.
- Blur / elevation: none.
- Typography: Helvetica / Inter / Arial (neutral grotesque). Tight leading on headings (~1.0–1.1), comfortable body leading (~1.5). Clear modular scale (e.g. 14 / 16 / 20 / 32 / 64).
- Weights: hierarchy via 400 / 700 (and 900 for display), not color or ornament.
- Spacing: strict baseline grid on an 8px base; multi-column grid with consistent gutters (e.g. 12 columns, 24px gutter).
